Hangarin: Task Manager
A simple task management web application built as a Django demonstration project. It’s designed to show how Django can be used to create a functional web app with features like adding, editing, and organizing tasks. The project focuses on learning the basics of Django models, views, templates, and forms while building something practical. Hangarin helps demonstrate how a full-stack web application works — from handling data in the backend to displaying it in a clean, user-friendly interface.
